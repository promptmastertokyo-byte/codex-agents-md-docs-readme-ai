# gif-animation: 差分領域限定のループGIF生成

同一構図の2枚(例: 目開き/目閉じ)から、差分のある領域だけを
クロスフェードさせるGIFを作る。フレームAを全面ベースに固定するため、
背景・髪・紙テクスチャは1ピクセルも動かない。

## 手順

1. **画像確認**: 2枚を `Read` で見て、同一構図であることと差分箇所
   (目・眉・頬・口など)を把握する
2. **領域決定**: 差分を覆う楕円を元画像座標 `cx,cy,rx,ry` で決める。
   位置が不確かなら crop して確認する:
   ```
   python3 -c "from PIL import Image; Image.open('A.png').crop((600,180,1020,520)).save('/tmp/check.png')"
   ```
   楕円は差分(眉〜口・両頬)を覆う最小限にする。髪の生え際や輪郭線は
   含めない
3. **生成**:
   ```
   pip install Pillow  # 未導入の場合
   python3 .claude/skills/gif-animation/scripts/face_anim_gif.py \
       frameA.png frameB.png outputs/animation/out.gif \
       --region 795,330,155,135 --hold-a 2000 --hold-b 3000
   ```
   - frameA = 始点(ループの基準)、frameB = 終点
   - `--hold-a/--hold-b` でホールド秒数を調整(ms)
   - 領域が複数あるときは `--region` を繰り返し指定
4. **検証**: ピークフレーム(index 3)を抽出して合成の破綻がないか目視。
   疑わしければ全フレームの min-max 差分でモーションマップを作り、
   動いているのが指定領域だけであることを確認する
5. **送付**: `SendUserFile`(display: render)でGIFを送る

## 禁止事項(過去の失敗パターン)

- **画像全体のクロスフェード禁止**: 2枚の紙テクスチャ・髪の微差が
  画面全体のチラつきになる。必ず領域マスクでブレンドする
- **バウンス・平行移動禁止**: 笑いの上下揺れ等を画像シフトで表現すると
  全体が揺れて見える
- **ワープによる髪揺れは既定でオフ**: 揺れゾーンに背景の直線構造
  (フェンス・トンボ)が入ると全体が動いて見える。求められた場合のみ、
  毛束輪郭に沿ったマスク+構造物除外で行う

## このリポジトリでの注意

- 出力先は `outputs/animation/`。`outputs/` は gitignore 済みなので
  生成物はコミットせず `SendUserFile` で渡す
- サイズ目安: 720px幅・6フレームで約1.5MB。大きくなる場合は共通
  パレット量子化(スクリプトに組込済)と幅の縮小で調整する
