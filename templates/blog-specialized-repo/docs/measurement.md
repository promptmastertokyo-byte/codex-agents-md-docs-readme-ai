# Measurement and Improvement

公開後は2〜4週間待ってから改善します。

## Metrics

- Search Console: 表示回数、クリック率、検索クエリ、平均掲載順位
- Analytics: 滞在時間、離脱、スクロール
- WordPress: コメント、内部リンク、更新日

## Improvement Rules

| 状態 | 改善 |
| --- | --- |
| 表示されるがクリックされない | タイトルとディスクリプションを直す |
| クリックされるが読まれない | 冒頭の結論と表を前に出す |
| 想定外クエリで出ている | 見出しとFAQを追加する |
| 順位が伸びない | 具体例、一次情報、内部リンクを増やす |

## Cadence

- 公開後2〜4週間: 初回確認
- その後月1回: 上位記事のみ改善
- 制度系記事: 更新があれば即確認
