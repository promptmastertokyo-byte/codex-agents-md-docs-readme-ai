# Blog Playbook

AI編集部の勝ちパターン、失敗パターン、運用ルールを蓄積する場所です。

## Operating Loop

```text
ネタ集め
↓
記事企画
↓
記事作成
↓
レビュー
↓
公開
↓
計測
↓
改善
↓
次のネタ集めに反映
```

## Title Patterns

- `2026年版｜[検索語]はいくら？[具体金額]の判断基準を[権威性]で解説`

## Intro Patterns

- 読者の悩みを1文で言う
- 150文字以内に結論を出す
- 数字・条件・対象読者を早めに見せる

## Improvement Rules

| Signal | Action |
| --- | --- |
| 表示回数が多くCTRが低い | タイトルとメタディスクリプションを直す |
| クリック後すぐ離脱 | 冒頭の結論、表、具体例を前に出す |
| 11〜30位で停滞 | FAQ、具体例、内部リンクを追加する |
| 想定外クエリで表示 | 新記事化または見出し追加を検討する |

- 制度系記事は、勝ちパターンより最新性と正確性を優先する。
