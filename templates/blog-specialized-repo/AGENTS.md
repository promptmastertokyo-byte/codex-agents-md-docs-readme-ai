# Blog Repo Agent Rules

## Response Style

- 日本語で返答
- 結論ファースト
- 変更したファイルと確認結果を短く報告

## Blog Rules

- `blog/style-guide.md` に沿って記事を書く
- 記事は `blog/drafts/` に置く
- 公開済み記事の改善案は `blog/published/` または `docs/measurement.md` の方針に沿って扱う
- 制度・税務・法律・医療・金融の記事は、公開直前に最新情報を確認する
- 公式情報、確認日、監修メモを記事内に残す

## Safety

- `.env` や認証情報を出力しない
- WordPressへの公開操作は事前確認する
- `git commit` は確認を取ってから実行する
